# Endless Login (Concept) 

Endless is a concept developed as experiment as a possible login procedure for social project. You can see presentation/demo on  [**behance.net**](https://www.behance.net/gallery/46337983/Endless-Login-(Concept)).

## Reasons

I belive that mostly more of us expirience some sort of frustration when we come across new web resource that seems useful for us, but yet requieres us to complete a registration in order to perform desired sort of actions.
